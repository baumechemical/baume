ข้อมูลผลิตภัณฑ์

สรุปคุณสมบัติและจุดควบคุมหลักของระบบซาตินนิเกิ้ล

  • ประเภท: ระบบซาตินนิเกิ้ล (Satin Nickel) ใช้ตัวพา Satin Carrier A/B และ Foaming Agent SM
  • ผิวสำเร็จ: ด้านเนียน ปรับได้ตั้งแต่กึ่งด้านถึงด้านจัด
  • ช่วงกระแสใช้งาน: 3–8 A/dm² (แนะนำ 4–6 A/dm²)
  • การใช้งาน: เครื่องประดับ ชิ้นส่วนอิเล็กทรอนิกส์ ฮาร์ดแวร์ งานตกแต่ง และรองรับโครเมี่ยมต่อ
  • วัสดุรองรับ: เหล็ก สแตนเลส ทองเหลือง/ทองแดง สังกะสีไดคาสต์* (เตรียมผิวเหมาะสม)
  • *งานบางชนิดควรมีคอปเปอร์สไตรค์หรือนิเกิ้ลสไตรค์ก่อน

การกวนโยกชิ้นงาน/กวนเชิงกล หลีกเลี่ยงลมตรง
การกรองกรองต่อเนื่อง หรืออย่างน้อยกรองทุกวันหลังงาน

รายการอัตราการใช้

Satin Foaming Agent SMตามหน้างาน หรือ ~0.02 ซีซี/ลิตร/ชั่วโมง

หมายเหตุด้านเคมี

  • นิเกิ้ลซัลเฟต: แหล่ง Ni หลัก ระดับต่ำเกินไปทำให้ความเป็นซาตินลดลง
  • นิเกิ้ลคลอไรด์: ช่วยการละลาย/เงาขึ้นได้ ไม่ควรเติมมากเกิน
  • กรดบอริก: ส่งผลต่อความคงตัวของผิวซาติน ควรรักษาในช่วงมาตรฐาน
  • Satin Carrier A/B: ช่วยให้ซาตินติดผิวดี หากเติมมากไปอาจเงาระยะสั้นได้ ตัวพาไม่หายด้วยการกรองเชือก/ผ้า
  • Foaming Agent SM: ตัวกำหนดความด้าน เติมมาก→ด้านมาก ปรับตามเวลา/กระแส/อุณหภูมิ

แนวทางแก้ปัญหาเบื้องต้น (Satin Nickel)

ผิวไม่เรียบ/ไม่สม่ำเสมอ

  • ปรับกระแสให้อยู่ 4–6 A/dm² และคุมอุณหภูมิให้คงที่
  • เพิ่มการกวนชิ้นงาน/การไหลเวียน ลดจุดอับ

รอยขีด/รอยคลื่น

  • ขัดเตรียมผิวและล้างให้สะอาด ลดสิ่งปนเปื้อน
  • กรองต่อเนื่องและเปลี่ยนสื่อกรองตามรอบ

สี/ความด้านไม่สม่ำเสมอ

  • ทวนอัตราส่วน Carrier A/B และ Foaming Agent
  • คุม pH 4.1–4.5 และอุณหภูมิ 50–60 °C

ด้านเกิน/เงาเกิน

  • ด้านเกิน → ลด Foaming Agent หรือกระแส/เวลา
  • เงาเกิน → เพิ่ม Foaming Agent ทีละน้อย และ/หรือปรับกระแสลง

การยึดเกาะไม่ดี

  • ตรวจขั้นตอนเตรียมผิว (ขจัดคราบ/แอซิดแอคติเวต)
  • ตรวจตะขอ/หัวล่อและการสัมผัสไฟฟ้า

ความทนทานเคลือบต่ำ

  • เพิ่มความหนาเวลา และรักษากระแสในช่วงแนะนำ
  • ทวนอุณหภูมิ/การกวนเพื่อขนาดเกรนที่เหมาะสม

*ควรสวม PPE เช่น ถุงมือ แว่นตานิรภัย และหน้ากาก ระหว่างกระบวนการ
