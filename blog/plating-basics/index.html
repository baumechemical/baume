คู่มือพื้นฐานงานชุบโลหะ: เคมี อุปกรณ์ และการดูแลรักษา

แผนผังภาพรวม → เคมีหลัก → อุปกรณ์สำคัญ → ค่าตั้งต้น → QC/ความปลอดภัย → ปัญหาพบบ่อย

สารบัญ

แผนผังกระบวนการ เคมีชุบและบทบาท ตารางค่าตั้งต้น อุปกรณ์สำคัญของไลน์ การควบคุมคุณภาพ/บำรุงรักษา ความปลอดภัยและสิ่งแวดล้อม ตารางปัญหาพบเจอบ่อย ลิงก์ที่เกี่ยวข้อง

แผนผังภาพรวมกระบวนการชุบ

จัดลำดับเพื่อลด drag-in/ปนเปื้อน และคุมคุณภาพชิ้นงาน

ล้างไขมัน→ ล้างน้ำ→ กัด/กระตุ้นผิว→ ล้างน้ำ→ ชั้นรองพื้น/Strike→ ชั้นชุบหลัก→ หลังชุบ (Passivation/Seal/Topcoat)→ ล้าง/อบแห้ง/ตรวจ

เคมีชุบหลักและบทบาท

  • Cleaners: ด่าง/อิมัลซิไฟ/อัลตร้าโซนิก เอาไขมัน/คราบ → ลดพิตติ้ง/ยึดเกาะแย่
  • Activation/Pickle: ละลายออกไซด์/ฟิล์มพาสสีฟ → เตรียมผิวก่อน strike/บ่อหลัก
  • Strike/Undercoat: เช่น Wood’s Ni, Acid Cu (non-cyanide) เพิ่มการยึดเกาะ/ครอบคลุม LCD
  • Main baths: Ni (Watts/Sulfamate), Zn (Acid/Alkaline), Cu (Acid), Sn/Ni/Cr ตามข้อกำหนด
  • Passivation/Conversion: ป้องกันหมอง/กัดกร่อน (เช่น โครเมต 3+ บนซิงค์)
  • Sealer/Topcoat/Anti-tarnish: เพิ่มทนทาน/คุมโทนสี/ความมัน
  • สารช่วย: brightener/leveler/carrier, wetting agent, grain refiner, inhibitors ฯลฯ

ตารางค่าตั้งต้น (อ้างอิงเริ่มทดลองบนชิ้นจริง)

ขั้นตอน พารามิเตอร์ตั้งต้น (ช่วง) หมายเหตุ
ล้างไขมันด่าง/อัลตร้าโซนิก 50–65°C, 2–8 นาที; กวน/อากาศพอเหมาะ ล้างหลายขั้น (Counter-flow) เพื่อลด drag-in
Activate/Pickle กรดเหมาะวัสดุ, 10–60 วิ (ควบคุมกัดผิว) ล้างทันทีหลังจุ่ม ลดคราบ/รอยไหล
Strike (ตัวอย่าง: Wood’s Ni / Acid Cu) 2–6 A/dm², 0.5–3 นาที ให้ทั่วก่อนบ่อหลักเพื่อยึดเกาะดี
Ni (Watts) 45–60°C; pH 3.8–4.5; Rack 2–8 A/dm² / Barrel 0.5–1.5 กรอง 5–10 µm ต่อเนื่อง; ถุงแอโนด
Zn (Acid chloride) 20–35°C; pH 4.8–5.6; 1–4 A/dm² ระวัง HCD ไหม้; เติม wetting agent
Passivation (Trivalent) อุณหภูมิ/เวลา ตามสเปก; จุ่ม 5–60 วิ ล้าง DI + อบแห้งลมร้อน
Sealer/Topcoat/Anti-tarnish DoR ตามชนิด (น้ำ/ทินเนอร์/ไฟฟ้า) ทดสอบการถู/หมอง/SST

หมายเหตุ: ใช้สเปกน้ำยาจริงเป็นหลัก ตารางนี้เป็นไกด์เริ่มต้น ปรับครั้งละตัวแปรและบันทึกผล

อุปกรณ์สำคัญของไลน์ชุบ & จุดตรวจ

อุปกรณ์ บทบาท จุดตรวจ/บำรุง
Rectifier/Busbar/สาย จ่ายกระแส/แรงดันเสถียร รั่วน้อย Ripple ต่ำ, หนีบแน่น/สะอาด, จุดต่อไม่ร้อน
แร็ค/ตะกร้า/บาร์เรล จับชิ้นงาน/ไฟสัมผัสดี ผิวสัมผัสสะอาด, ฉนวนส่วนที่ไม่ต้องชุบ
แอโนด+ถุง แหล่งโลหะ/สมดุลไอออน ถุงไม่ฉีก, สัดส่วนแอโนดต่อแคโทดเหมาะ
ปั๊ม/กรอง/เอดักเตอร์ คุมฟลูว์/ความสะอาด กรอง 5–10 µm, ซีลแน่น, ไม่มีดูดอากาศ
ฮีตเตอร์/คอยล์/ชิลเลอร์ คุมอุณหภูมิ เทอร์โมทำงาน, เซนเซอร์แม่น, ตะกรันต่ำ
ดูดไอ/สครับเบอร์ ลดไอกรด/ไอเกลือ ทิศลมถูกต้อง, กรอง/น้ำยาสครับพร้อม
ระบบล้าง/DI ลด drag-in/คราบ ล้างหลายขั้น, น้ำนิ่งสะอาด, ค่าความนำไฟฟ้าต่ำ
บำบัดน้ำเสีย ตามกฎหมาย/สิ่งแวดล้อม pH/ตกตะกอน/สารช่วยตก ตรงสเปก
Hull cell/แผ่นทดสอบ วิเคราะห์พฤติกรรมบ่อ ทดสอบเป็นรอบ, เก็บภาพ/บันทึกแนวโน้ม

การควบคุมคุณภาพ/การดูแลรักษา

  • ทุกกะ: เช็ค T/pH/การนำไฟฟ้า/ฟลูว์; เติมน้ำ/สารตามสเปก
  • รายวัน: กรองต่อเนื่อง, วัดความหนา/ยึดเกาะ (cross-hatch/tape), ตรวจสีโทน
  • รายสัปดาห์: วิเคราะห์สารสำคัญ/โลหะปน; Carbon treatment เมื่ออินทรีย์สะสม
  • รายเดือน: Hull cell / สอบเทียบเครื่องมือ / ตรวจถุงแอโนด/ฮีตเตอร์
  • ตามสเปก: ทำ Salt Spray Test / การถู / ความทนหมอง
  • บันทึกแนวโน้มทั้งหมด → ใช้คาดการณ์/ป้องกันก่อนเกิดปัญหา

ความปลอดภัยและสิ่งแวดล้อม (SHE)

  • PPE ครบ: แว่น/หน้ากากกันไอเคมี/ถุงมือ/เอプロน
  • ระบายอากาศดี + สครับไอกรด/ไอเกลือ
  • จัดเก็บสารเคมีแยกประเภท มีรองรับหกรั่ว + SDS ติดใกล้จุดใช้งาน
  • ชุดอุดรอยรั่ว/ดูดซับพร้อมใช้งาน และซ้อมแผนฉุกเฉิน
  • บำบัด/กำจัดของเสียตามกฎหมาย บันทึกการขนย้าย

ตารางปัญหาพบเจอบ่อย

อาการ สาเหตุหลัก แนวทางแก้เบื้องต้น
พิตติ้ง/รูเข็ม น้ำมัน/ฝุ่น, กวน/เปียกผิวไม่พอ, ฟองก๊าซติด ปรับกรอง/ฟลูว์, เติม wetting agent, ปรับตำแหน่งชิ้นงาน
ไหม้/รัน (HCD) กระแสสูง, ระยะขั้วไม่เสมอ, over-agitation เฉพาะจุด ลด A/dm² ทำ ramp, ใส่ shield/ปรับฟิกซ์เจอร์
ผิวหม่น/เงาไม่มา T ต่ำ, pH หลุด, additive ต่ำ/เสื่อม, โลหะปน เพิ่ม T/จูน pH, เติม/รีเฟรช additive, carbon+dummy
ยึดเกาะไม่ดี/ลอก Clean/Activate ไม่พอ, ข้าม strike, ผิว passive ย้อนทำความสะอาด/activate, ทำ strike ให้ทั่ว
โทนสี/Passivation เพี้ยน เวลา/อุณหภูมิ/ pH หลุด, รีเฟรชไม่พอ คุม T/เวลา/pH ตามสเปก, เติม/รีเฟรชอ่าง
