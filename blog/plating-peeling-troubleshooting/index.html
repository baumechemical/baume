ชุบลอก/ยึดเกาะไม่ดี: สาเหตุหลัก & วิธีแก้

ตั้งค่าตั้งต้น → เตรียมผิว → หน้าต่างกระบวนการ → ฟิกซ์เจอร์ → ดูแลบ่อ → ตารางอาการ–สาเหตุ–แนวทางแก้

สารบัญ

ค่าเริ่มต้นที่แนะนำ การเตรียมผิวก่อนชุบ หน้าต่างกระบวนการ การจัดวาง/ฟิกซ์เจอร์ การดูแลบ่อ & คุณภาพ ตารางปัญหาพบเจอบ่อย ลิงก์ที่เกี่ยวข้อง

ค่าเริ่มต้นที่แนะนำ

ใช้เป็นจุดตั้งต้นเพื่อ “ให้ติดก่อน” แล้วค่อยปรับละเอียดตามชิ้นงานและสเปกน้ำยาจริง

วัสดุ ชั้นรองพื้น/วิธี ช่วงตั้งต้น หมายเหตุ
เหล็กกล้า (คาร์บอนต่ำ/ทั่วไป) Wood’s Nickel Strike → บ่อหลัก (Ni/Zn/Cu ตามสเปก) Strike 2–6 A/dm², 45–55°C, 0.5–3 นาที ล้าง HCl สั้นๆ ก่อนเข้า Strike เพื่อตัดออกไซด์
เหล็กแรงสูง (HRC ≥ 40) เหมือนเหล็กทั่วไป + ชุบเสร็จอบไล่ไฮโดรเจน Bake 190–220°C, 2–4 ชม. (ตามมาตรฐานลูกค้า) ทำภายใน 1 ชม.หลังชุบ ลดความเปราะไฮโดรเจน
สเตนเลส 304/316 Wood’s Nickel Strike (จำเป็น) → บ่อหลัก Strike 2–6 A/dm², pH ~1, 45–55°C, 1–3 นาที ต้องสดใหม่/สะอาด คุมเวลาให้ทันทีหลัง Activate
ทองเหลือง/ทองแดง Acid Cu strike หรือ Ni strike → บ่อหลัก 1–3 A/dm², 20–30°C, 0.5–2 นาที Desmut/Activate ให้สะอาด ลดฟิล์มซัลไฟด์
อะลูมิเนียม (5xxx/6xxx) Etch → Desmut → Zincate (2 ครั้ง) → Cu/Ni strike Double zincate 30–90 วิ/ครั้ง, ล้าง DI ทุกขั้น อย่าข้าม Double zincate เพื่อลดลอกหลังชุบ
สังกะสีไดคาสท์ (Zamak) Degas 120–150°C → Cu strike (ตามระบบ) → บ่อหลัก Strike อ่อน 0.5–2 A/dm², 20–30°C, 0.5–2 นาที ก่อนชุบควรอบไล่ก๊าซ ลดพอง/โพรงอากาศ

หมายเหตุ: ยึดสเปกน้ำยาจริงเป็นหลัก ตารางนี้เป็นไกด์เพื่อเริ่มให้ติดก่อน แล้วค่อยจูนทีละตัวแปร

การเตรียมผิวก่อนชุบ

  • ล้างไขมัน/ทำความสะอาดหลายขั้น (ด่าง/อิมัลซิไฟ/อัลตร้าโซนิก) → ลดน้ำมัน/ฝุ่นที่ทำให้ลอก
  • กล/พ่นทราย/ขัด (ถ้าสเปกรับได้) เพื่อเพิ่มพื้นที่ยึดเกาะและกำจัดผิวเสีย
  • Activate/Pickle ตัดออกไซด์ให้เหมาะวัสดุ (เช่น HCl บนเหล็ก, ผสมฟลูออไรด์บนสเตนเลส, Desmut บน Al)
  • Al: Double zincate เสมอ (จุ่ม–ลอก–จุ่มใหม่) แล้วค่อยเข้า Cu/Ni strike
  • Zamak/Porous: อบไล่ก๊าซก่อนชุบ ลดพอง/ฟองดันจนลอก
  • ล้างน้ำหลายขั้นแบบ counter-flow + DI ขั้นสุดท้าย ลด drag-in ที่ทำให้ผิวปนเปื้อน

หน้าต่างกระบวนการ (ที่กระทบต่อการยึดเกาะ)

ขั้น/บ่อ ตัวแปรสำคัญ เหตุผล/ผลกระทบ
Clean/Rinse T, เวลา, กวน/อากาศ, คุณภาพน้ำล้าง คราบหลงเหลือ = ผิว passive → ลอก/ยึดเกาะต่ำ
Activate/Pickle ชนิดกรด/pH, เวลา, อุณหภูมิ น้อยไป = ออกไซด์คงอยู่, มากไป = ผิวกัดเกิน/คราบสเกล
Strike A/dm², เวลา, T, pH, ความสดของบ่อ ตั้งอ่อนเกิน = ไม่ครอบคลุม LCD; แรงเกิน = ไหม้/รันหลุด
Main plating A/dm², Throwing power, ฟลูว์/กรอง, additive การกระจายไม่ดี → บาง/หนาไม่สม่ำเสมอ → หลุดจุดอ่อน
Post/Passivation เวลา/T/pH โครเมต/ซีล, การอบ/อบไล่ H เวลาเกิน/สภาวะไม่ตรง → ฟิล์มอ่อนตัว/ยึดเกาะตก

การจัดวาง/ฟิกซ์เจอร์

  • จุดสัมผัสไฟต้องสะอาด/แน่น กระแสไม่ตัน ไม่เกิดจุดร้อน
  • จัดทิศกระแสให้ครอบคลุม LCD (เพิ่ม robbers/shields ถ้าจำเป็น)
  • เว้นระยะชิ้นงาน/แอโนดสม่ำเสมอ ลด HCD ไหม้–LCD บาง
  • หลีกเลี่ยงโพรงดักอากาศ—เจาะรูระบาย/ปรับองศา/การกวน
  • Rack vs Barrel: Barrel ใช้ A/dm² ต่ำกว่า, เพิ่มเวลาคลุมทั่ว

การดูแลบ่อกัด/ชุบ & คุณภาพ

  • กรองต่อเนื่อง 5–10 µm, เปลี่ยนไส้กรองตามชั่วโมงใช้งาน
  • Carbon treatment เมื่ออินทรีย์สะสม/ผิวหม่น/ยึดเกาะตก
  • Dummy plate เพื่อตัดโลหะปน/สารรบกวน
  • ถุงแอโนดสะอาด/ไม่ฉีก; สัดส่วนแอโนด:แคโทดเหมาะ
  • ทดสอบยึดเกาะ: cross-hatch/tape, bend, thermal cycle; บันทึกแนวโน้ม

ตารางปัญหาพบเจอบ่อย

อาการ สาเหตุราก วิธีแก้/แนวทาง
ลอกทันทีหลังชุบ/หลังล้าง Clean/Activate ไม่พอ, ข้าม/strike ไม่ทั่ว, ผิว passive ย้อนทำความสะอาด/activate, เพิ่ม/ปรับ strike ให้คลุม LCD
พอง/ลอกหลังทำ Passivation/SST เวลาหรือ pH โครเมตหลุดสเปก, ล้างน้ำไม่ดี, ไฮโดรเจนฝัง คุม T/เวลา/pH, ล้าง DI หลายขั้น, Steel แข็ง: อบไล่ H หลังชุบ
ลอกเฉพาะ LCD Strike อ่อนเกิน/เวลาสั้น, ฟลูว์นิ่ง, จุดบอดกระแส เพิ่มเวลา strike, ปรับฟลูว์/ทิศกระแส, ใส่ robbers/shields
ไหม้/ลอกเฉพาะ HCD/ขอบคม A/dm² สูง, ระยะขั้วไม่สม่ำเสมอ, ฟิกซ์เจอร์ชี้ไฟแรง ลดกระแสหรือ ramp, ปรับระยะ, ใช้ shields/round edges
ลอกหลังอบ/ประกอบ ก๊าซใน Zamak/Al ดันผิว, ฟิล์ม passivation อ่อน Pre-bake ชิ้นก่อนชุบ, Double zincate (Al), จูน passivation
ลอกบนสเตนเลส/พื้นที่เชื่อม ข้าม Wood’s Ni, Activate ไม่พอ, ฟิล์ม Cr–oxide หนา ทำ Wood’s Ni ใหม่ทันทีหลัง activate, เพิ่ม wetting/ฟลูว์
ลอกบนทองเหลือง/คราบแดง ซัลไฟด์/ออกไซด์ผิว, ล้างน้ำ/Desmut ไม่ดี Desmut ให้หมด, จัดล้าง DI, ใช้ strike ที่เหมาะแล้วค่อยบ่อหลัก
ลอกเป็นแผ่น (underfilm) คราบน้ำมันแฝงจาก forming/สลิปผิว เพิ่มขั้นทำความสะอาด/ตัวทำละลาย/อัลตร้าโซนิก, ตรวจน้ำล้าง
